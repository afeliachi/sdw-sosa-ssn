System Capabilities Module

The System Capabilities Module imports SSN and adds classes and properties required for the description of the operating conditions and capabilities of systems, in particular Sensors and Actuators.

The namespace for the System Capabilities Module is http://www.w3.org/ns/ssn/systems/
The suggested prefix for the system capabilities, operating ranges, and survival ranges terms is ssn-system

An RDF graph for this is available.

The SSN-System graph is available at http://www.w3.org/ns/ssn-system/.

System Capabilities was a module in the 'Horizontal Segmentation' section of the previous edition of the SSN Ontology [[vocab-ssn-20171019]]. This module has received little attention or testing, so it has been moved to Appendix in this edition.

ssn-system:hasOperatingConditionKind

IRI: http://www.w3.org/ns/ssn/systems/hasOperatingConditionKind

an OWL Object Property

has operating condition kind — Relation between an ObservationCollection (or Observation) and an OperatingConditionKind describing the kind of the operating conditions in the context of the system. A short but extensible list of operating condition kinds is provided in SSN-System, such as normal operating conditions, suboptimal conditions, and survivable conditions.

The System Capabilities Module includes a short but extensible list of operating condition kinds:

ssn-system:NormalOperatingConditions, ssn-system:SuboptimalConditions, ssn-system:SurvivableConditions

IRI: http://www.w3.org/ns/ssn/systems/NormalOperatingConditions

an OWL Individual

IRI: http://www.w3.org/ns/ssn/systems/SuboptimalConditions
